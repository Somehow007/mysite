## 二、问题诊断与解决

### 2.1 发现的问题

#### 问题1: Elasticsearch索引分片不可用
- **现象**: ES集群状态为red，blog_articles索引分片无法分配
- **原因**: 索引使用了ik_smart分词器，但ES未安装IK分词器插件
- **解决方案**: 
  1. 删除原有索引
  2. 修改ArticleDocument和UserDocument，使用standard分词器替代ik_smart

#### 问题2: 日期格式转换错误
- **现象**: `Unable to convert value '2025-12-12T21:00:47' to java.util.Date`
- **原因**: ES中存储的日期格式与Spring Data Elasticsearch的默认格式不匹配
- **解决方案**: 
  1. 在ArticleDocument中添加日期格式注解 `@Field(type = FieldType.Date, format = DateFormat.date_optional_time)`
  2. 删除旧索引，重新创建并同步数据

#### 问题3: 搜索API空关键词查询失败
- **现象**: 当keyword为空时，`findByTitleContaining("")`无法正确返回结果
- **解决方案**: 修改ArticleServiceImpl，当keyword为空时使用`findAll()`方法
